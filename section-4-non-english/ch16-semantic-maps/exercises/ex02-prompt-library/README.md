# Exercise 16.2: Build a Prompt Library

## Goal
Create a library of pre-built prompts in your native language for common development tasks.

## Instructions

1. Identify 10 tasks you do regularly with AI tools
2. Write an effective English prompt for each
3. Translate each prompt to your native language
4. Test both versions — does the native version produce comparable results?
5. Refine the translations using glossary terms from Exercise 16.1
