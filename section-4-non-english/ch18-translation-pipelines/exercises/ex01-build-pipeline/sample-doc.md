# Getting Started with AcmeClaw

## Installation

Install AcmeClaw using npm:

```bash
npm install -g acmeclaw
acmeclaw onboard
```

## Configuration

Create a steering doc at `~/.claude/CLAUDE.md` with your preferences. This file tells the AI agent who you are and how you work.

## Your First Agent

AcmeClaw agents run on Amazon Bedrock using the Working Backwards methodology. Define what done looks like, write the tests, then let the agent build.

See the [PR/FAQ template](/templates/prfaq/) to get started.
