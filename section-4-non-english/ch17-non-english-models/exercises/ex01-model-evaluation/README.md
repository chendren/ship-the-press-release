# Exercise 17.1: Model Evaluation for Your Language

## Goal
Evaluate 2-3 models' performance for coding tasks in your native language.

## Instructions

1. Choose 2-3 models (e.g., Claude, GPT-4, Gemini, DeepSeek)
2. Give each the same 5 coding tasks in your native language
3. Score each on: correctness, code quality, understanding of intent
4. Document which model performs best for your language
