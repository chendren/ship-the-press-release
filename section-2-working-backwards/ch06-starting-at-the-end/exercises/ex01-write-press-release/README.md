# Exercise 6.1: Write Your Press Release

## Goal
Write a press release for a product or feature you want to build — before writing any code.

## Instructions

1. Choose something you actually want to build (not hypothetical)
2. Use the [PR/FAQ template](/templates/prfaq/press-release-template.md)
3. Write as if the product is shipping today
4. Be specific: numbers, pricing, capabilities, availability
5. Run the [quality checklist](/templates/prfaq/prfaq-checklist.md) against your result

## The Test
Show your press release to someone who doesn't know your project. Can they explain what you're building? If not, rewrite.
