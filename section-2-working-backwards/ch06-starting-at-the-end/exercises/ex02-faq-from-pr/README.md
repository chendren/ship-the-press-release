# Exercise 6.2: The FAQ That Stress-Tests Your PR

## Goal
Write the FAQ that a skeptical customer and an honest engineering team would produce.

## Instructions

1. Take your press release from Exercise 6.1
2. Write 6+ customer questions — start with the hardest objection
3. Write 3+ internal questions — include the one engineering doesn't want to answer
4. Use the [FAQ template](/templates/prfaq/faq-template.md)
5. Every answer must include specific details — no "contact us for more information"
