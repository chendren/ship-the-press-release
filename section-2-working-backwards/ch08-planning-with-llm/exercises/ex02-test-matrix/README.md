# Exercise 8.2: Test Matrix

## Goal
Create a comprehensive test matrix from your build plan.

## Instructions

1. Take your build plan from Exercise 8.1
2. Use the [test matrix template](/templates/build-plan/test-matrix-template.md)
3. Generate tests for every requirement
4. Include: unit, integration, edge cases, and failure modes
5. Assign priorities (P0/P1/P2)
