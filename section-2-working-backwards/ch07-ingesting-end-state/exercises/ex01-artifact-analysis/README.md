# Exercise 7.1: Artifact Analysis

## Goal
Analyze the sample PR/FAQ to extract explicit requirements, implicit requirements, open questions, and contradictions.

## Instructions

1. Read `sample-artifact.md`
2. Use the [analysis template](/templates/artifact-analysis/analysis-template.md)
3. List all explicit requirements (what it says directly)
4. List all implicit requirements (what it assumes)
5. List open questions (what it doesn't address)
6. Identify any contradictions
