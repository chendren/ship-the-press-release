# Exercise 7.2: Build a Research Tool

## Goal
Build a tool that ingests PR/FAQ artifacts and produces structured analysis — using the starter artifact collection.

## Instructions

1. Review the sample artifacts in `artifacts/`
2. Build a CLI tool that reads a PR/FAQ markdown file and outputs an analysis
3. The analysis should include: requirements, implicit assumptions, open questions
4. Test it against each sample artifact

## Starter Artifact Collection

The `artifacts/` directory contains 5 sample PR/FAQs across different product categories. Your research tool should work on all of them.
