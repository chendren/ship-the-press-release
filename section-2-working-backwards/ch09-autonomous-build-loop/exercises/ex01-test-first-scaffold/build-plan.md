# Feature: String Utilities

Build a module with these functions:

1. `slugify(text: string): string` — Convert text to URL-safe slug ("Hello World" → "hello-world")
2. `truncate(text: string, maxLength: number): string` — Truncate with "..." if over limit
3. `wordCount(text: string): number` — Count words in text
4. `capitalize(text: string): string` — Capitalize first letter of each word
