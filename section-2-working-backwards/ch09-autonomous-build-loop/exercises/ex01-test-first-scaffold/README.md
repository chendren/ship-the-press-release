# Exercise 9.1: Test-First Scaffold

## Goal
Write a complete test suite for a feature — then let the agent implement it to pass the tests.

## Instructions

1. Read `build-plan.md` — a simple feature spec
2. Write tests in `expected/tests/sample.test.ts` that cover every requirement
3. Verify: all tests fail (nothing implemented yet)
4. Feed the tests + build plan to your AI coding tool
5. Let it implement until all tests pass

## The Point

The tests ARE the specification. When they pass, the feature is done. No ambiguity.
