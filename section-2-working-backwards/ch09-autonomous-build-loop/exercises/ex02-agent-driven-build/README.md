# Exercise 9.2: Agent-Driven Build

## Goal
Give the agent a complete specification (steering doc + build plan + test matrix) and let it build autonomously.

## Instructions

1. Create a steering doc for a small project
2. Create a build plan with phased tasks
3. Write the test matrix
4. Feed all three to your AI coding tool
5. Let it build. Don't intervene unless it asks.
6. Count: How many iterations until all tests pass?
