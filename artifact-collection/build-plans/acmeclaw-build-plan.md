# AcmeClaw Build Plan

See [Appendix J](/appendices/appendix-j-building-acmeclaw/) for the complete walk-through.

## Phases

1. **Foundation**: Theme overlay, branding, S3/CloudFront infrastructure
2. **Website**: Static site with services, book, author sections
3. **Pre-order Backend**: API Gateway + Lambda + DynamoDB
4. **Managed OpenClaw**: AgentCore deployment architecture
5. **Enterprise Features**: SSO, audit logging, vetted skill registry
