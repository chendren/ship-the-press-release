# Exercise 2.1: First Spec

## Goal
Write a specification for a small feature, then build it from the spec. Compare the experience to vibe coding (Exercise 1.1).

## Instructions

1. Fill out `spec-template.md` for the same CLI tool from Exercise 1.1 (URL fetcher/summarizer)
2. Feed the completed spec to your AI coding tool as context
3. Build the tool from the spec in a single pass
4. Compare: How many iterations did spec coding require vs. vibe coding?
