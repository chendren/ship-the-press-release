# Exercise 1.1: First Vibe

## Goal
Build a working CLI tool using only conversational prompts — no specification document, no architecture diagram, no test plan.

## Instructions

1. Open your AI coding tool
2. Use the prompt in `starter.ts` as your opening message
3. Let the conversation flow naturally — describe what you want, evaluate the output, iterate
4. Stop when you have something that works
5. Note: How many iterations did it take? Where did the model guess wrong?

## What to Observe

- How quickly the model produced something functional
- Where its assumptions diverged from your intent
- The point where iteration stopped improving and started cycling
- Whether you could articulate *why* it was wrong (this is the seed of spec coding)

## Compare

After completing this exercise, compare your experience against `expected/output.md`.
