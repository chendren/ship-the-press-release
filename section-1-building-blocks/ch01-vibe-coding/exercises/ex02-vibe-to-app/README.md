# Exercise 1.2: Vibe to App

## Goal
Build a complete small application using pure vibe coding. Track every prompt and response to see the full conversation arc.

## Instructions

1. Choose a small app idea (timer, todo list, calculator, etc.)
2. Start with a single conversational prompt
3. Build the entire app through iteration — no pre-planning
4. Keep a log: copy each prompt you sent and summarize each response
5. Count: total prompts, successful iterations, wasted iterations

## Deliverable

A working app + your conversation log in `expected/output.md` format.

## Reflection Questions

- What percentage of your prompts were "corrections" vs. "additions"?
- At what point (if any) did you wish you had a specification?
- Would a different developer produce the same app from the same starting prompt?
