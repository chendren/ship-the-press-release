# Exercise 5.1: First Skill

## Goal
Create a skill that teaches an AI agent how to perform a specific task in your domain.

## Instructions

1. Choose a task you do regularly (code review, deployment, data migration, etc.)
2. Fill out `starter-SKILL.md` with the knowledge the agent needs
3. Install it in your OpenClaw/AcmeClaw instance
4. Test: Does the agent use it when you ask about that task?

## The Key Insight

The `description` field in the frontmatter determines whether the agent activates the skill. Invest more time in the description than in the body.
