# Exercise 3.1: First Steering Doc

## Goal
Write a project-level CLAUDE.md for a real project you're working on, then measure the difference it makes.

## Instructions

1. Choose a real project (or use the URL summarizer from Ch 1-2)
2. Start with `starter-CLAUDE.md` — fill in every section
3. Open your AI coding tool with the steering doc in place
4. Ask the model to make a change to your project
5. Compare: Is the output closer to what you wanted vs. without the steering doc?

## Measurement

Before and after, rate these 1-5:
- How often did the model match your coding style?
- How often did it follow your conventions?
- How often did it need correction?
