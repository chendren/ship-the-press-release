# Exercise 3.2: Your First Hook

## Goal
Create a hook that automatically enforces one of your coding rules.

## Instructions

1. Pick a rule from your CLAUDE.md (Exercise 3.1)
2. Write a hook that enforces it automatically
3. Test: Does the hook catch violations? Does it allow valid code?

## Example Rules to Enforce

- No `console.log` in production code
- No files larger than 200 lines
- No TODO comments without a linked issue
- All new functions must have a test

## Starter

Use `starter-hook.ts` as your template.
