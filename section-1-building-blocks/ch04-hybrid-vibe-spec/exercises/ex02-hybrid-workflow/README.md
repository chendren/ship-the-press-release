# Exercise 4.2: Hybrid Workflow

## Goal
Build a feature using both modes: vibe for exploration, spec for implementation.

## Instructions

1. **Phase 1 (Vibe)**: Spend 10 minutes vibing a rough version of a "bookmark manager CLI" — save URLs with tags, search by tag
2. **Phase 2 (Transition)**: Look at what you built. Write a spec for the version you actually want.
3. **Phase 3 (Spec)**: Rebuild from the spec. Compare the two versions.

## Deliverable
Both versions (vibe and spec) side by side, with notes on what changed.
