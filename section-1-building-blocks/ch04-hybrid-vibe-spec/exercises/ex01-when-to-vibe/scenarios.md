# Mode Classification Scenarios

For each scenario, decide: Vibe, Spec, or Hybrid?

1. **Exploring a new API**: You've never used the Stripe API. You want to see how checkout sessions work.

2. **Building a login page**: Your app needs email/password auth with a JWT token system.

3. **Prototyping a dashboard**: The PM wants to see "something" by tomorrow to get stakeholder feedback.

4. **Fixing a timezone bug**: Users in Australia see the wrong date. You know the root cause is in `formatDate()`.

5. **Adding CSV export**: Users need to export their data as CSV. The data model has 15 fields with specific formatting rules.

6. **Refactoring a 500-line function**: It works but it's unmaintainable. You need to split it into modules.

7. **Creating a marketing landing page**: The founder described it verbally over lunch.

8. **Implementing rate limiting**: The API needs to limit requests per API key with configurable thresholds per plan tier.

9. **Writing tests for existing code**: 200 lines of untested business logic.

10. **Building a Slack bot**: "It should respond to /summarize and give a summary of the channel's last 24 hours."
