# Improved Prompts — What to Look For

Your improved prompts should be:
- **Shorter** — remove context the steering doc already provides
- **More specific** — "implement X using Y" not "build something"
- **Structured** — input format, output format, constraints
- **Honest about scope** — "only modify these 2 files" not "update the app"

The best prompt improvement is often deletion.
