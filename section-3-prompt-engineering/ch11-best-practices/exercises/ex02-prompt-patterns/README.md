# Exercise 11.2: Prompt Patterns

## Goal
Apply the three most effective prompt patterns to a real task.

## Instructions

1. Choose a real development task
2. Apply each pattern:
   - **Constraint pattern**: "Do X. Do NOT do Y."
   - **Example pattern**: "Here's an example of what I want: [example]"
   - **Structured output pattern**: "Return JSON with these fields: [schema]"
3. Which pattern produced the best result for your task? Why?
