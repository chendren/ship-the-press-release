# Templates

Reusable templates referenced throughout *Ship the Press Release*. These are the artifacts you produce before writing code.

| Template | Purpose | Book Reference |
|----------|---------|----------------|
| [PR/FAQ](prfaq/) | Press release + FAQ for a product that doesn't exist yet | Ch 6, Appendix C |
| [Build Plan](build-plan/) | Architecture and implementation plan from PR/FAQ analysis | Ch 8, Appendix C |
| [Artifact Analysis](artifact-analysis/) | Structured analysis of end-state artifacts | Ch 7, Appendix C |
| [Project State](project-state/) | Session continuity and project handoff | Appendix H |

## How to Use

1. Copy the template to your project
2. Fill in the sections — the placeholders guide you
3. Feed the completed artifact to your AI coding tool as context
4. The quality of your build is proportional to the clarity of these artifacts
