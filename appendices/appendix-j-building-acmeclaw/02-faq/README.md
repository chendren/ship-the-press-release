# Step 2: Write the FAQ

The FAQ stress-tested the press release by asking the hardest customer and internal questions.

## Key Questions That Shaped the Product

- "Can I leave?" → No lock-in, standard Markdown/YAML export
- "What about CVE-2026-25253?" → Sandbox limits blast radius
- "How does pricing work for multi-agent?" → Per-agent unit model
