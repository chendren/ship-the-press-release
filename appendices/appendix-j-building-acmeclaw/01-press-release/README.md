# Step 1: Write the Press Release

Before any code existed, the AcmeClaw press release was written. It defined: what the product is, who it's for, three concrete capabilities, and specific pricing.

See [acmeclaw-prfaq/press-release.md](/acmeclaw-prfaq/press-release.md) for the complete artifact.

- Product name and tagline
- Target customer (teams, not solo developers)
- Three testable capabilities (sandbox, registry, ops layer)
- Pricing tiers ($49, $149, Enterprise)
- Availability statement
