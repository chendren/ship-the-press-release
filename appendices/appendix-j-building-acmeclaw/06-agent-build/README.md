# Step 6: The Agent Build

With steering docs, build plan, and tests in place, the RALPH Loop executed:

1. **Read** — PR/FAQ + analysis loaded
2. **Analyze** — Requirements extracted (explicit + implicit)
3. **Layout** — Build plan generated with ordered phases
4. **Produce** — Code generated to pass tests
5. **Harmonize** — Tests run, failures addressed, iterate

The agent built until the tests passed.
